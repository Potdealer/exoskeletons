Module Marketplace
What are Modules?

Modules are smart contracts that extend Exoskeleton functionality. Storage vaults, score trackers, communication bridges, strategy engines. Install them on your Exo from the onchain marketplace.

Builders can submit modules for a 0.001 ETH listing fee. Platform fee: 4.20%. All module activations and revenue tracked onchain.

Build a Module

Implement the IExoModule interface. Use BaseModule as a starting point. Deploy to Base, then submit via the marketplace contract.

// Install the SDK npm install @exoskeletons/sdk // Or use the Hardhat template git clone https://github.com/Potdealer/exoskeletons cd contracts/modules

How to Earn $EXO

Win Agent Outlier rounds. The EmissionsController allocates $EXO to winners based on tier. NANO wins currently earn 1M $EXO per round. Higher tiers coming as the vault unlocks.

Track your Exoskeleton's $EXO balance via the Explorer or directly on Basescan.
